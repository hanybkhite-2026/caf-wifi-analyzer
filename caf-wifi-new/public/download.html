📡 CAF-WIFI Local Agent

Download and run on your laptop to enable real WiFi scanning on any device

🪟 Windows (Recommended)

Single file — just download and double-click. Node.js required.

⬇ Download .bat (Windows) ⬇ Also get agent.js
Step 1: Download caf-wifi-agent.bat
Step 2: Double-click to run
Step 3: Agent shows your IP (e.g. 172.16.46.148)
Step 4: In app → Connect Agent → enter:
    http://172.16.46.148:7788/api/scan
⚠️ If Windows blocks it: right-click → Properties → Unblock → OK

🐧 Linux / 🍎 macOS

Download agent.js and run with Node.js.

⬇ Download agent.js
Step 1: Download caf-wifi-agent.js
Step 2: Open terminal in download folder
Step 3: node caf-wifi-agent.js
Step 4: Enter URL shown in app → Connect Agent

📱 Mobile (Android / iPhone)

Run the agent on your laptop, then connect your phone via the app.

Step 1: Run agent on laptop (see above)
Step 2: Phone must be on SAME WiFi as laptop
Step 3: Open app on phone
Step 4: Tap "Connect Agent" button
Step 5: Enter: http://LAPTOP-IP:7788/api/scan
Step 6: Real WiFi scan on your phone! ✅
Your laptop IP from ipconfig (Windows) or ip addr (Linux).
Example from your network: http://172.16.46.148:7788/api/scan
Need Node.js? Download free from nodejs.org → LTS version
