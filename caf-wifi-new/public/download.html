Android App (APK) NATIVE
Scans WiFi directly — no laptop needed
Step 1: Download the APK below
Step 2: On your phone: Settings → Security → Unknown sources → Allow
Step 3: Open the downloaded APK → Install
Step 4: Open CAF-WIFI → grant location + WiFi permissions
Step 5: Tap Scan → see all real networks! ✅
⬇ Download Android APK
⚠️ APK must be built first. Run bash build-android-apk.sh in Firebase Studio terminal, then deploy.

Windows Agent
Run on your laptop — phone connects to it
Step 1: Download both files to same folder
Step 2: Double-click caf-wifi-agent.bat
Step 3: Note your IP (e.g. 172.16.46.148)
Step 4: In app → Connect Agent → enter: http://IP:7788/api/scan
