# **App Name**: NetPulse CAF

## Core Features:

- Real-time CAF-WIFI Scanner: Continuous scanning and discovery of enterprise CAF networks with signal strength color coding.
- Dynamic Dashboard Visualizations: Integrated charts and status cards for network health, client distribution, and interference monitoring.
- AI Network Optimizer Tool: A tool using generative AI to analyze scan patterns and suggest configuration changes to reduce interference.
- Firestore Network Central: Centralized data persistence for scan logs, location tracking, and team metadata.
- Enterprise Export Module: Instant generation of network health reports in PDF and CSV formats for management reviews.
- Admin Fleet Management: Control panel to manage team members, scan quotas, and organization-wide network performance stats.
- Analytics Performance Suite: Longitudinal data tracking of speed test results and signal reliability over weekly cycles.

## Style Guidelines:

- The primary color is a vibrant electric blue (#3B82F6) representing network connectivity and technical precision.
- The background follows a sophisticated dark mode palette utilizing a deeply desaturated navy (#11141A).
- The accent is an analogous cyan (#06B6D4), providing high-contrast highlights for buttons and active states.
- A pairing of 'Space Grotesk' for technical headlines and 'Inter' for dense data tables ensures both clarity and a high-end feel.
- Linear, minimal SVG icons that pulse or change color based on real-time signal health indicators.
- A multi-tab dashboard structure with cards that utilize glassmorphism for a modern, transparent enterprise aesthetic.
- Smooth transitions between dashboard tabs and subtle breathing effects on the signal intensity radar chart.
